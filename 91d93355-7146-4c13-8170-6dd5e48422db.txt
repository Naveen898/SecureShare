About the Role 	
Position Tile: Cyber Security Intern
Title: Intern
Reporting to: CISO
Location: Bengaluru

 			










Job Profile

Purpose of the Job 
The Cyber Security Staff is responsible for protecting the organization's digital assets, minimizing security threats, and ensuring the confidentiality, integrity, and availability of data. They will work closely with the IT and Development teams to identify vulnerabilities, implement security measures, and create and implement security procedures and guidelines. They will also be responsible for staying up to date with the latest security trends and technologies and continuously assess and improve the organization's security posture.
Primary Responsibilities

* Assist in securing systems, networks, endpoints, and cloud environments (Public/Private Cloud Platforms), while gaining hands-on exposure to encryption, identifying security vulnerabilities, supporting vulnerability management, ensuring compliance and applying secure-by-design practices.
* Involvement in core network security technologies and practices, including:
o Firewalls, proxies, VPN, Intrusion detection/prevention systems (IPS/IDS).
o DDoS protection and defense strategies against large-scale attacks.
o Cloud security and access controls (CASB), email and web security gateways, authentication measures (MFA), and endpoint/rogue device protection.
o Commonly used network security tools and platforms such as Cisco, Palo Alto, Zscalar.
* Gain hand-on exposure to Identity & Access Management (IAM) concepts, including:
o Privileged Access Management (PAM), Directory services (Azure AD, Active Directory, user and application onboarding/offboarding, and common identity integration standards and protocols, such as SAML, OAuth, OpenID Connect, and SCIM.
* Exposure to cybersecurity tools and their lifecycle management, including installation, configuration, and maintenance tasks. 
* Gain Knowledge and Hands-on experience with SIEM and SOAR tools for use case development, security monitoring, Incident response, and administration.
* Collaborate with the development team to integrate security measures into the Software Development Life Cycle (SDLC).
* Develop and implement security policies, procedures, and guidelines to ensure a robust security posture across the organization.
* Ability to Identify and promptly respond to security incidents, perform root cause analysis and assist in implementing mitigation measures.
* Develop, maintain and execute security playbooks, runbooks, scenario-based guidelines for effective incident response.
* Stay up to date with the latest security trends, technologies, and emerging threats to continuously enhance security postures.
* Apply knowledge of threat management, threat intelligence, and threat hunting to proactively identify and mitigate security risks.
* Skill is scripting, automation, and coding for security management, with familiarity in leveraging AI and Machine learning for Cybersecurity considered an added advantage.



Candidate Profile:
* Students currently pursuing B. Tech/B. E in Cyber Security / Computer Science or a related discipline. 
* Cyber Security related certifications (if any) will be an added advantage.
* Cyber security projects or other relative experience in information and Cyber Security
* Exposure to Automation, scripting (Python, PowerShell, bash etc.), or security related projects as part of academic curriculum is a plus.
* Actively participated in Cybersecurity completions, hackathons, and relevant industry events, with a track record on winning or earning a recognition.  

Education & Professional Qualifications:
Personal Skills: 
* Excellent problem-solving skills.
* Strong analytical and critical thinking abilities.
* Effective communication skills for collaborating with team members and stakeholders.

Technical Skills:
* Basic knowledge of cybersecurity principles and practices.
* Familiarity with operating systems, including Windows and Linux.
* Understanding of the software development life cycle (SDLC) and its security implications.
* Knowledge of security fundamentals, including cryptography, authentication, and authorization.
* Familiarity with vulnerability testing and assessment tools.
* Understanding of cybersecurity tools, such as SIEM, IDS/IPS, antivirus, and intrusion detection systems.
* Familiarity with creating runbooks, playbooks, security guidelines, and procedures.
* Exposure to network security tools such as firewalls, routers, WAF, proxy, DDoS, and IAM systems.

A strong understanding of cybersecurity principles and technologies is essential.
